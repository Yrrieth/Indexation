Fichtre

Le projet www.zoolemag.com consiste à produire une vitrine numérique au magazine Zoo. Zoo est un média web et papier dans le domaine de la bande dessinée, du manga et des comics, à la fois agrégateur et éditeur de contenu de qualité.

Le but de ce projet est de créer un site proposant un support numérique pour un ensemble d'article provenant du magazine papier sans pour autant garder sa structure d'origine, ceci afin de garder une fluidité et un confort pour la navigation des utilisateurs. Le site proposera également du contenu inédit du support papier afin d'inciter les utilisateurs à visiter le site pour leur offrir une complémentarité au magazine.

L'accès au site se fera via l'interface web pour tous les utilisateurs. Un accès spécial sera dédiée aux rédacteurs pour la rédaction d'article et un autre accès spécial pour les administrateurs du site et du serveur. Une page spéciale ou un filtrage par machine devra être mis place pour distinguer les accès de chaque utilisateur.
